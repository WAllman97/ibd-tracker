# IBD Tracker — App Features

# Core MVP Features

## 1. Daily Symptom Logging

Users can log:
- pain
- urgency
- bowel movements
- blood presence
- fatigue
- stress
- sleep
- meals
- medication
- notes

Goal:
- under 2 minutes per entry

---

## 2. Dashboard

Visual overview of:
- symptom trends
- averages
- flare periods
- weekly summaries
- recent entries

---

## 3. Trend Charts

Charts for:
- pain over time
- stress over time
- sleep vs symptoms
- bowel movement frequency
- fatigue trends

---

## 4. Daily Notes

Free text notes for:
- unusual symptoms
- meals
- stressful events
- medication changes
- doctor updates

---

## 5. Weekly Summary

Automatically generated weekly overview.

Example:
- average pain
- best/worst days
- symptom trend
- sleep average
- possible triggers

---
